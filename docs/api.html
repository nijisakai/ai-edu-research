数据API文档

本API提供28个JSON数据集的RESTful访问接口，支持跨域请求。所有数据基于1,690个AI教育典型案例的系统分析。

API密钥

所有数据请求需要API密钥验证

默认密钥：BNUSLI

Base URL
https://nijisakai.github.io/ai-edu-research/data/

使用示例

JavaScript

const API_KEY = 'BNUSLI';
const BASE = 'https://nijisakai.github.io/ai-edu-research/data/';

async function fetchData(filename) {
  const resp = await fetch(BASE + filename);
  if (!resp.ok) throw new Error('请检查API密钥');
  return resp.json();
}

// 获取省份分布数据
const provinces = await fetchData('province_distribution.json');
console.log(provinces);

Python

import requests

BASE = 'https://nijisakai.github.io/ai-edu-research/data/'
API_KEY = 'BNUSLI'

def fetch_data(filename):
    resp = requests.get(BASE + filename)
    resp.raise_for_status()
    return resp.json()

# 获取工具分布数据
tools = fetch_data('tool_product_distribution.json')
print(f"Top工具: {tools['top30_tools'][:5]}")
