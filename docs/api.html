数据 API 文档

所有分析数据以 JSON 格式提供，可直接通过 HTTP GET 请求获取。支持跨域(CORS)访问。

Base URL

https://nijisakai.github.io/ai-edu-research/data/

使用示例

// JavaScript
fetch('https://nijisakai.github.io/ai-edu-research/data/province_distribution.json')
  .then(r => r.json())
  .then(data => {
    console.log(data.case_count_by_province);
    // { "浙江省": 432, "四川省": 190, ... }
  });

# Python
import requests
data = requests.get('https://nijisakai.github.io/ai-edu-research/data/metadata.json').json()
print(f"总记录数: {data['total_rows']}, 案例数: {data['unique_cases']}")

可用数据集

metadata.json

1.1 KB

数据集元信息：总行数(3,815)、唯一案例数(1,690)、列名列表

province_distribution.json

1.3 KB

30省份案例数量与工具数量分布

stage_distribution.json

0.7 KB

学段(小学/初中/高中/幼儿园)分布统计

subject_distribution.json

8.9 KB

319种学科分布统计及学科-工具映射

tool_product_distribution.json

5.8 KB

AI工具使用频次Top30、产品形态、产品分类

scenario_analysis.json

3.3 KB

应用场景L1/L2层级分析（助学/助教/助育/助评/助管）

framework_stats.json

2.7 KB

理论框架标注统计：三赋能、iSTAR、数字教学法、智慧教育境界、创新深度评分，含交叉表

nlp_clusters.json

24.4 KB

10个NLP语义聚类：聚类大小、关键词及TF-IDF权重

nlp_all_results.json

229.7 KB

综合NLP分析：TF-IDF关键词、LDA主题模型、共现网络、技术路径

case_deep_analysis.json

64.2 KB

46个典型案例深度分析：iSTAR评级、五育分类、工具使用、教学创新评估

causal_analysis_results.json

73.8 KB

因果与统计分析：对应分析、MCA、序数/多项Logistic回归、路径分析、SHAP

deep_insights.json

43.0 KB

深度洞察挖掘：工具生态分析、数字鸿沟、创新指标、教师创造性模式

company_market_share.json

1.2 KB

企业市场集中度数据（CR5/CR10/HHI）

api-index.json

自动生成

所有数据文件的完整索引，含文件大小信息
