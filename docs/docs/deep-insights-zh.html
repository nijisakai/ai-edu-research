  1. [反直觉] 经济水平与创新深度的Spearman相关系数 r=-0.272, p=0.1876 — 经济水平与创新深度呈负相关——欠发达地区反而创新更深
  2. [学段差异] 技术代际最高学段: 高中 (均值=3.43), 最低: 幼儿园 (均值=3.18)
