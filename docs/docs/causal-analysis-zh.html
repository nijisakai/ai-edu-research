核心发现总结

  1. 对应分析（CA）：学科 x 应用场景的对应分析表明，前两个维度捕获了总惯量的88.8%（维度1=50.6%，维度2=38.2%）。德育和美育等学科远离主流"智能辅导"极点，而STEM学科则聚集在"助学"（学习辅助）附近。学段 x 赋能类型的对应分析显示出较强的结构特征（总惯量=0.088），其中幼儿园与中学阶段明显分离。

  2. 多重对应分析（MCA）（N=3,548）：三个潜在维度解释了总惯量的36.9%。维度1对比了罕见的机构赋能（学校层面、HUM）与主流的学生赋能 + 高级协同极。维度2将教师赋能和制度变革与面向学生的模式分离。维度3捕捉了评价/评估导向与教学导向之间的差异。

  3. 有序逻辑回归（N=3,548，伪R2=0.201）：自研产品具有最强效应（OR=33.7, p<0.001），其次是iSTAR层级（OR=9.9, p<0.001）。三赋能分类中"赋能教师"（OR=0.09）和"赋能评价"（OR=0.07）相较于赋能学校，与较低的创新深度相关联。比例优势假设被违反（p<0.001），表明效应在不同创新评分阈值间存在差异。

  4. 多项逻辑回归（伪R2=0.290）：在预测iSTAR层级时，技术代际是最强的区分变量（HMC相对于参照组的RRR=0.25）。D1（深度学习）和D3（循证教学）具有相反的效应：D1=0.14（强烈降低HMC的发生概率），D3=0.13（降低HMC的发生概率），表明高级协同需要深度学习与循证教学方法的共同支撑。

  5. 路径分析（N=3,815，R2=0.107）：技术代际对创新深度具有显著的总效应（beta=0.041, p=0.001）。D4（人机互信）路径起正向中介作用（间接效应=0.008, Sobel z=2.81, p=0.005, Bootstrap置信区间不包含零）。值得注意的是，D3（循证教学）路径起负向中介作用（间接效应=-0.019, Sobel z=-6.05, p<0.001），原因在于更高的技术代际实际上与更低的D3得分相关（beta=-0.047），表明新兴技术尚未充分整合循证教学实践。

  6. 地理不平等分解：不平等主要存在于省内而非省际之间。就创新深度而言，泰尔指数方差的98.1%来自省内（基尼系数=0.197）。不平等程度最高的省份为湖南（基尼系数=0.251）、辽宁（0.240）和河北（0.227）。区域间不平等仅占0.3%-0.5%，表明AI教育领域的数字鸿沟是地方性的而非区域性的。

  7. 聚类画像（K=10）：所有连续变量的方差分析均呈现大效应量。D4（人机互信）具有最高的区分力（eta2=0.989, F=38,359），其次是D3（eta2=0.778）和D1（eta2=0.739）。创新深度的eta2=0.455。分类变量卡方检验显示，技术代际（Cramer's V=0.393，中等效应）和三赋能分类（V=0.340，中等效应）对聚类的区分效果最佳，而学段的区分力较弱（V=0.110，小效应）。

  8. 关联规则挖掘（443条规则，提升度范围[0.66, 6.12]）：提升度最高的规则（lift约6.1）一致地将"助教"（教学辅助）场景与"赋能教师"及HMC(1)协同层级相关联。小学 + 赋能教师的组合几乎确定性地指向"助教"场景（置信度=0.987）。这揭示了教师导向产品与中等协同设计之间的紧密耦合关系。

  9. 交互效应分析：学段 x 赋能类型对iSTAR层级的影响是最具实质性的模型（R2=0.286），其中仅赋能类型就解释了32.1%的方差（eta2=0.321，大效应）。区域 x 技术代际的交互效应在统计上显著（p=0.033）但效应量较小（eta2=0.029），表明不同区域在技术代际与创新之间的映射关系存在差异。学段 x 技术代际的交互效应不显著（p=0.137）。

  10. 预测建模（随机森林，5折交叉验证准确率=66.3%，Cohen's Kappa=0.767）：自研状态是最重要的预测变量（重要性=0.221, SHAP=0.087），其次是D3循证教学（重要性=0.139, SHAP=0.064）和iSTAR层级（重要性=0.078, SHAP=0.045）。SHAP值与基尼重要性的排名高度一致，证实了过程导向特征（D维度得分、协同层级）在预测创新深度方面优于结构性特征（区域、技术代际）。
